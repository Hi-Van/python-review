# Variables

```python
# with variables
character_age = "18"
character_name = "Van"

print("There was once a man named " + character_name + ", he was " + character_age)

# without variables
print("There was once a man named Van, he was 18")

# variable examples
string = "this is a string"
integer = 8
float_example = 2.0
boolean = True
```
